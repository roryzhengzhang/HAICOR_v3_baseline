Operation introduction

You will see a user interface as below:

This interaction zone is dragable and zoomable, and it allows you to perform the following basic operations:

Create new node (concept):

You can create a new node by right clicking anywhere on canvas except nodes and edges

Modify exisitng node (change its name):

You can modify an existing node by right clicking it and selecting "modify node"

Delete node:

You can delete an existing node by right clicking it and selecting "delete node"

Create new edge between concepts:

You can create a new edge by right clicking the starting node and selecting "create link". Then you need type in the name of ending node in the pop-up

Delete edge:

You can delete an existing edge by right clicking it and selecting "delete edge"

Modify exisitng edge (change its relation type):

You can modify an existing edge by right clicking it and selecting "modify relation type"
